4  ডেটা ম্যানিপুলেশন

ডেটা বিশ্লেষণ করার জন্য ডেটাকে পরিষ্কার ও সাজানো অত্যন্ত গুরুত্বপূর্ণ। R-এ ডেটা ম্যানিপুলেশনের জন্য dplyr প্যাকেজটি ব্যাপকভাবে ব্যবহৃত হয়।

4.1 dplyr প্যাকেজ

প্রথমত, আমাদের প্যাকেজটি ইনস্টল ও লোড করতে হবে:

# install.packages("dplyr") # প্রথমবার হলে এটি রান করতে হবে
library(dplyr)

Attaching package: 'dplyr'
The following objects are masked from 'package:stats':

    filter, lag
The following objects are masked from 'package:base':

    intersect, setdiff, setequal, union

4.2 filter()

filter() ফাংশন ব্যবহার করে ডেটা ফ্রেম থেকে নির্দিষ্ট শর্ত অনুযায়ী সারি (Rows) বেছে নেওয়া যায়।

# একটি ডেমো ডেটাসেট তৈরি
data <- data.frame(
  Name = c("A", "B", "C", "D"),
  Age = c(20, 25, 30, 22),
  Score = c(85, 90, 75, 88)
)

# বয়স ২৫ এর কম এমন রেকর্ডগুলি ফিল্টার করা
young_people <- filter(data, Age < 25)
print(young_people)
  Name Age Score
1    A  20    85
2    D  22    88

4.3 select()

select() ফাংশন ব্যবহার করে ডেটা ফ্রেম থেকে নির্দিষ্ট কলাম বেছে নেওয়া যায়।

# শুধুমাত্র Name এবং Score কলাম নির্বাচন করা
name_score <- select(data, Name, Score)
print(name_score)
  Name Score
1    A    85
2    B    90
3    C    75
4    D    88

4.4 mutate()

mutate() ব্যবহার করে নতুন কলাম তৈরি বা পুরোনো কলামের মান পরিবর্তন করা যায়।

# Score-এর সাথে ৫ যোগ করে নতুন কলাম তৈরি
new_data <- mutate(data, New_Score = Score + 5)
print(new_data)
  Name Age Score New_Score
1    A  20    85        90
2    B  25    90        95
3    C  30    75        80
4    D  22    88        93

4.5 summarize()

summarize() ব্যবহার করে ডেটার সারাংশ (Summary) তৈরি করা যায়, যেমন গড়, যোগফল ইত্যাদি বের করা।

# গড় Score বের করা
summary_data <- summarize(data, Average_Score = mean(Score))
print(summary_data)
  Average_Score
1          84.5
